# 后端 (服务端)
1. 定义多个接口:
    - npm i koa-router 安装路由，用来分门别类的定义后端接口地址
    - 分类：
        - 路由层：定义接口地址，例如 /api/auth/login 
        - 控制层：处理路由层定义的接口，例如调用登录函数
        - 模型层：与数据库交互，例如查询用户信息
    - npm i koa-bodyparser 辅助 koa 解析 post 请求体中的参数
